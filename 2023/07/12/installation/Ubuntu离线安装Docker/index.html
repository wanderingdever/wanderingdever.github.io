Ubuntu离线安装Docker

下载

下载路径：https://download.docker.com/linux/ubuntu/dists/xenial/pool/stable/amd64/

下载版本

离线安装docker需要下载3个包，containerd.io ，docker-ce-cli，docker-ce
containerd.io_1.2.6-3_amd64.deb
docker-ce_19.03.9_3-0_ubuntu-xenial_amd64.deb
docker-ce-cli_19.03.9_3-0_ubuntu-xenial_amd64.deb

安装

下载完毕后拷贝到ubuntu上用 dpkg 命令安装，安装顺序：先安装 containerd.io 跟 docker-ce-cli，最后安装docker-ce，命令：sudo
dpkg
-i xxxx.deb依次执行安装，顺序不能改变

启动

systemctl start docker
systemctl status docker

设置自启动

# 设置 docker自动启动
systemctl enable docker.service
# 查看docker 在系统中状态
systemctl list-unit-files |grep docker
