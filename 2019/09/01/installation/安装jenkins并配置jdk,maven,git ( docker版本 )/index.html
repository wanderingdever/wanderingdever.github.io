安装jenkins并配置jdk,maven,git ( docker )

前置条件先安装好docker,docker-compose,jdk,maven,git在宿主机上

拉取镜像

docker pull jenkinsci/blueocean:1.24.8

创建挂载文件目录

mkdir /home/jenkins_home

编写docker-compose.yml

启动docker-compose

docker-compose up -d

打开jenkins安装相关插件

git, maven
